📊 Smart Data Analyst Dashboard

An interactive Streamlit-based Data Analysis Dashboard that allows users to upload datasets and perform exploratory data analysis (EDA) with visual insights in just a few clicks.

🚀 Features
📁 Upload CSV dataset
📊 Dataset overview (rows, columns, null values)
📈 Interactive visualizations
🔍 Data filtering & exploration
📉 Missing value analysis
📊 Automatic charts & insights
⚡ Fast and simple UI for non-technical users

🛠️ Tech Stack
Frontend: Streamlit
Backend: Python
Data Handling: Pandas, NumPy
Visualization: Plotly

⚙️ Installation & Setup

-cd smart-data-analyst-dashboard

2️⃣ Create Virtual Environment

-python -m venv venv

-venv\Scripts\activate   # Windows
3️⃣ Install Dependencies

-pip install -r requirements.txt

4️⃣ Run Application

-streamlit run app.py

🧠 How It Works

-User uploads dataset (CSV)
-Data is loaded using Pandas
-Automatic data profiling is done
-Visualizations are generated
-Insights are displayed in Streamlit UI
