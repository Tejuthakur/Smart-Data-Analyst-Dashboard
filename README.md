# 📊 Smart Data Analyst Dashboard

An interactive AI-powered data analysis dashboard that helps users explore datasets, generate insights, and visualize data easily.

---

## 🚀 Features
- Dataset preview (rows, columns, missing values)
- Data visualization (charts)
- AI-generated insights
- Data quality analysis
- AI question answering system

## 🖼️ Screenshots
- Dashboard view
- Visualization charts
- AI insights output
- Data quality report

- AI API
